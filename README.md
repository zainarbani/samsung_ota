# Samsung OTA
Android OMA-DM client app with FUMO capabilities designed for retrieving firmware updates and their metadata from Samsung's official servers.

## Code
- This app is heavily adapted from [SamsungFumoClient](https://github.com/timschneeb/SamsungFumoClient) & [KeyAttestation](https://github.com/vvb2060/KeyAttestation).
- Also heavily coded by AI :).
